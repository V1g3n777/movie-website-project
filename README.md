## 📖 Description

This project is a modern movie discovery web application built with **Vanilla JavaScript**, **HTML5**, and **CSS3**, using the **TMDB (The Movie Database) API** to retrieve real-time movie data.

The application allows users to search for movies, browse popular and trending titles, explore detailed movie information, and filter content by genres. All data is fetched dynamically through REST API requests and rendered using JavaScript DOM manipulation without relying on front-end frameworks.

The main purpose of this project was to strengthen my understanding of JavaScript fundamentals, asynchronous programming, DOM manipulation, API integration, and responsive web development while building a real-world application.

## ✨ Features

- Search movies by title
- Browse popular and trending movies
- Filter movies by genres
- View movie ratings, release dates, and overviews
- Display high-quality posters and backdrops
- Responsive design for desktop and mobile devices
- Dynamic content rendering using JavaScript
- Real-time data fetched from the TMDB API
- Fast and intuitive user interface
